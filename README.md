DESCRIPCION:
    Este proyecto simula una competencia de cocina donde distintos participantes son evaluados por jueces en varias rondas.
    El programa procesa los puntajes y calcula estadísticas como:
        -Puntaje total
        -Rondas ganadas
        -Mejor ronda
        -Promedio de puntajes

INSTALACION DE DEPENDENCIAS:
    1. Clonar o descargar el repositorio.
    2. Crear un entorno virtual.
    3. Activar el entorno virtual.
    4. Instalar las dependencias.

EJECUCION DEL PROGRAMA:
    1. Abrir Jupyter Notebook.
    2. Abrir el archivo: ejercicio10.ipynb.
    3. Ejecutar todas las celdas en orden.
    4. Visualizar los resultados. 

TECNOLOGIAS UTILIZADAS:
    1. Python 3.
    2. Jupyter Notebook.
